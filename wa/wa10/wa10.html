Choose a Joke Category

My extension was to allow the user to choose what joke category they wanted. I did this by creating a new form with a select that has all the endpoint options that I provide from my chosen API. Then in JS, I had a map of the endpoint labels and the actual API endpoint and I made it so there is a const that corresponds to the select tag I created in HTML, and finding the value of that select and grabbing the API endpoint from the map is the endpoint that I actually access for the joke.

I'd want to use YouTube's API so that I could access videos and channels on YouTube and their metadata. I would use endpoints that find the number of views for a video, the number of subscribers to a channel, the genre/topic/tags that the video is under, and more. I would use this information to find videos to display to a user while the user is watching their current video.
